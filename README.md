This repository documents my hands-on journey through Database Management System lab experiments and foundational SQL codes. It features practical implementations of core database concepts, focusing on designing, querying, and managing relational databases efficiently using MySQL. It serves as a personal reference for backend data architecture.

The repository is organized chronologically by experiment. Each dedicated folder contains the specific SQL scripts executed during that session, alongside brief explanations of the applied concepts such as data manipulation, complex joins, subqueries, and normalization techniques.

To utilize these codes, you will need a local or remote installation of MySQL Server. Simply clone the repository, navigate to the desired experiment directory, and execute the .sql files within your preferred database client or command-line interface to observe the database operations in action.
